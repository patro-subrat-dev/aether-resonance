System Architecture & Technical Stack

Aether Resonance showcases several advanced browser features implemented natively without heavy framework wrappers:

  • Web Audio Synthesizer: Dynamic generation of synthesiser voices. Custom ADSR envelope mapping, high-resonance filter sweeps, bandpass feedback delay lines, and a custom synthetic impulse response reverb algorithm.
  • Vector Physics Engine: High-performance 2D physics running in an HTML5 canvas render loop. Simulates elastic body collisions, Newton-based gravitational orbital curves, and boundary-line segment deflections.
  • Sensory / Haptic Loop: Unifies the Device Vibration API (for physical tactile patterns on mobile) with auditory sub-bass thuds and visual canvas aberrations, mimicking physical mass impact.
  • Design Language: Retro Cyberpunk UI crafted with modern CSS glassmorphism, glowing CSS filter dropshadows, customized range-sliders, and responsive flex grid architectures.
